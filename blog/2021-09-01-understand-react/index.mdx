## 前言

前端技術不斷地在迭代，JavaScript 從 ES6 持續發表到 ES10，Vue 也從學生時期熟悉的 Vue 2 (Options API) 更新到了陌生的 Vue 3 (Composition API)，近一年接觸的 React 也從一開始碰到的 16 更新到 18 了。

至於我為什麼這次要上這堂線上課程呢？

主要是因為 Vue.js 即將從 Vue 2 更新到 Vue 3 了，而我自己先看了一下 Vue 3 推出的 Composition API 之後，覺得內容跟觀念與 React 滿相似的，所以打算在 Vue 3 正式推出前先學習 React 好了，反正我也是遲早要學的，而且這樣能看的工作比較多 `( ˘•ω•˘ )`

除了課程內容之外，我也會持續整理之後遇到與 React 相關的觀念到筆記中，讓自己之後複習起來能更有效率。

## React Hooks

在開始之前，先聲明本書內容主要是以 Functional Component 與 Hooks 為主，只有一個小節會提到 Class Component。

### Why Hooks

- 共用重複的邏輯
- 可以獨立地被測試
