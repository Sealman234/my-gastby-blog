## 二、技術

### JavaScript

為了加強本身對 JavaScript 底層知識的理解，我看了 JavaScript: Understanding the Weird Parts 這門線上課程，並且整理了一系列的筆記，目前已經整理到 [Sealman's JavaScript Notes](https://javascript.sealman.tw) 這份文檔裡面。

看完之後對於底層有更多的瞭解，這些觀念也會幫助理解各種 JavaScript 的語法，對我的幫助還滿大的！

### Vue.js

今天因為幫公司導入 Vue CLI 取代 Gulp 的開發環境，特別研究了 Vue CLI MPA 模式的配置方式，同時也導入 Vue.js Style Guide 與 ESLint 等規範工具，希望在 Vue 2 這個部分能讓同事以 Best Practice 進行開發。

### React

因為不喜歡 Vue 3 改成這樣，個人還是愛用 Vue 2 的緣故，我選擇投奔 React 的懷抱，因此看了 React - The Complete Guide (incl Hooks, React Router, Redux) 這門線上課程，並且也有整理一本系列筆記，歡迎到 [Sealman's React Book](https://react.sealman.tw) 這份文檔去做查看喔。

不得不說，React 的學習曲線真的讓人感覺比較陡峭一點，不過只要硬爬過去，後面就趨緩很多了，對於自己能堅持下去我也感到不可思議，我到底是多賭爛 Vue 的改版 🫢

## 三、部落格

今年把部落格從 Hexo 改到 Gridsome 之後，現在又改用 Gatsby 架設。其實 Gridsome 就用得挺順手的了，只是因為剛好在學 React，所以想說乾脆連部落格都用 React 重構好了。

今年原本要發的文章都整理到兩個文檔專案裡面了，所以部落格看起來很冷清，但其實把文章一篇一篇拆出來，內容是很豐富的呢！

另外也很開心有收到讀者的回饋，雖然我都只是要寫給自己看而已，但是能幫助到其他人我還是覺得很榮幸，之後會繼續努力把關一下文章品質的。

## 六、電影

### 脫稿玩家 (Free Guy)

看到是萊恩雷諾斯主演，加上看過預告，男主角戴上眼鏡的開放世界長得超像 GTA 跟黑街聖徒，所以很期待正片的畫面與特效會如何呈現。

而且因為我本身很愛玩遊戲，所以看到出現各種彩蛋的時候都很嗨，雖然這部片不像一級玩家那樣整個搬進來，不過還是很有趣！

整體來說滿好看的，就算不玩遊戲的人也能看得很開心 😆
