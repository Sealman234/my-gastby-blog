## 關於我

### 學歷

就讀南部國立大學資訊管理學系，修習學校的通識課程與系上開設的一門前端課程後，才發現自己對前端領域的興趣，便開始上網查找更多學習資源，開啟前端自學之路。

### 自學方式

我是跟著 [**六角學院**](https://www.hexschool.com/) 的課程學習的，從最基礎的 HTML、CSS 開始，一路學習 jQuery、RWD、JavaScript、Bootstrap，與前端框架 Vue.js。

至於 Git 與 GitHub 我是使用到了，才會上網找資料，多找幾次大概就知道基本的使用了 XD

## 面試分享

### 阿利恩股份有限公司

來自日本與台灣的新創科技公司，雖然說是新創公司，但是好像已經在日本耕耘許久了，主要的工作內容是開發客製化的專案，大多以日本客戶為主。

這次與 CEO 面試，有交換聊聊許多想法，對方也不吝嗇地點出我可以再加強的技術與方向，感覺對技術很重視呢！

不過因為公司能開始實習的時間是從 4 月初，這樣我會太晚開始實習，所以最後只好婉拒哩。

### APPX 時賦科技

完成過許多專案，開發團隊擁有豐富的實戰經驗，主要吸引我的點是他們對程式教育的理念，畢竟我本身也是一個透過線上課程學習的學生，所以對於線上學習這種方式算是滿有經驗的。

團隊 Leader 很親切，可以感受到他對於團隊成員們的信賴與照顧，最後有順利錄取，不過因為公司性質是接案，而我比較希望可以開發自己的應用，比較嚮往那種跟著專案一起成長的感覺，所以最後我也是婉拒了。

### TAAZE 讀冊生活

讀冊的面試機會是六角的老師幫忙介紹的，面試前就有聽過這間，二手書買賣的部分是讀冊的主要經營項目，雖然我自己是屬於習慣去茉莉二手書店翻翻看實體書的那種人 XD

面試我的是開發部門的主管，他有提到因為老師推薦，公司才破例開出實習缺，對我的印象也比較深刻。

不過面試時，面試官大多只針對作品的 UI 美醜給予反饋，反而沒提到前端領域的技術問題，讓我覺得有些奇怪。而且對方不具備 UI/UX 專業知識，卻主觀地批評作品介面，讓我心情有點悶啊，畢竟這作品也經過六角 UI 設計的考驗了呢。

工作內容是開發接下來要推出的新服務，大概就是把二手書買賣做得更全面吧！

雖然面試過程讓我滿不開心的，但是這邊薪資待遇優於前兩間，加上我比較想要做自家產品，所以最後選擇投入這邊。

## 結語

在面試過程中，三家公司的面試官都有提到技術部落格的部分，也都認為這是很好的習慣。

雖然當初寫技術部落格，只是想進一步確保自己有吸收到知識，但最近發現部落格也有幫助到其他正在學習前端的人，更沒想到這次還能讓我在面試時與面試官更有話聊，覺得開心～
