## 一、工作

今年年初到讀冊實習，年中轉為正職，待的部門是一個公司內部的新創團隊，公司環境很好，同事都很 Nice 😊

雖然我覺得團隊在定義需求以及開發的流程都還有待加強，但我認為剛開始就先學習，把自己當作一塊海綿盡力吸收各種知識，好的知識吸收起來，爛的默默無視即可，畢竟有些人事物是我們無法改變的。

開始工作的這十個月，除了熟悉前端技術之外，也學習到跨部門的知識，溝通能力也有提高了，實習確實能提早瞭解業界的環境與技術，不枉我提早脫離學生身份進入職場啊！

## 二、技術

今年研究的技術範圍沒有很廣，大致上就是工作流程與前端方面的技術：

- 熟練 Vue.js、Vue Router、Vuex 語法與使用方式
- Trello 任務進度管理工具
- 透過 Vue.js Style Guide (Recommended) + ESLint 規範團隊的程式撰寫風格
- 研究 Quasar Framework 各個元件、PWA、SSR、SEO 等
- 優化程式寫法，例如：元件化、OOCSS、BEM 等
- 試用 Hexo 以外的靜態部落格架設工具，包含 VuePress、Jekyll、Gridsome、Gatsby、Hugo

除了 Vue.js，不知為何我自己喜歡用的幾個平台都是使用 React 居多，讓我也想要試試看 React 寫起來的感覺，因此在今年年底開始學習 React，希望新的一年也可以試著找 React 的工作。

另外 SEO 的需求也讓我之後想接觸 Next.js，同時也覺得自己的前端底子不夠深厚，而開始加強 JavaScript，此外像是效能調教、程式架構、單元測試等技能我也還需要補足，之後應該會需要做一個技能總整理。

## 三、鐵人賽

今年很心動想要參加鐵人賽，原本想以 Quasar Framework 為主題參賽，但寫了幾天之後覺得自己的文章質量不足，最後選擇忍痛放棄。
明年我一定要捲土重來！

## 四、部落格

滿開心這個部落格還是有人看到並給予回饋的。寫部落格是在 2019 年我開始學習前端不久時，大學的教授廖洧杰（也是六角學院校長）建議我可以將一路上所學到的技術，或是遇到的問題與其解法寫成文章，以便重新整理思緒、溫故知新，而且往後也可以當作求職的作品，豐富自己的履歷。

不過我有些文章沒有講得很仔細，有些可能是因為工作而研究（但是自己沒有很愛用的技術），或是我也喜歡偶爾寫一點生活文，加上我的文筆沒有很好，希望之後可以寫得更好。

同時因為自己有經營這個部落格，我也開始關注 SEO 的做法，搞得我對 SEO 也是有點興趣 XD

## 五、健康

在三月之前我還過著大學生活，可以直接去學校的羽球場、游泳池、健身房運動，幾乎都是免費使用的，有些則要負擔費用，但還是相當便宜。

離開大學來到台北工作後，運動場地的收費價格總是讓我望之卻步，不過幾個月調整心態後比較可以接受了，之後會試著找一些室內場館。對我而言，規律的運動習慣會讓我覺得在寫 Code 的時候思考邏輯比較清晰 (๑•̀ㅂ•́)و✧

## 七、美劇

分享一些今年看的美劇，關於影集的話都是在 Netflix 上看的。

首先，今年我終於離開「觀看盜版」的生活，正式訂閱 Netflix 囉！畢竟只要負擔一些費用就可以觀看到高畫質的影集，這麼好用當然要支持一下。另外我也開始使用環保餐具、環保吸管，也比較關心政治與經濟等議題了。

### 絕命毒師（Breaking Bad）

今年訂閱 Netflix 後馬上把它從第一季看到第五季，覺得還滿好看的，Walter 跟 Jesse 兩位主角的演技都很好，劇情步調偏慢，每位角色的性格能夠一步步地發展醞釀。

## 八、電影

今年比較忙，只有去電影院看一部電影而已。

### 無聲（The Silent Forest）

身為一位台南人，其實我是知道這部電影的故事背景的，因此我在觀看之前就已經做好心理準備，因為這會是一部較為沉重的電影。實際觀賞後，這部電影確實有壓迫感，看完之後會覺得滿沉重的，但我認為這個議題是重要而且需要被討論的，我很感謝導演與劇組對於這個議題的用心。
