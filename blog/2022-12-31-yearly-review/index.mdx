## 一、工作

今年離開樂屋進入 Funliday 與 Wistron，使用技術也從 Vue 轉換到 React。

去年年底開始，我利用下班與假日看 Udemy 的 React 課程，不過因為常常在休息耍廢打電動，所以拖拖拉拉地花了總共五個月的時間，才把 React 生態圈的基本技能都掌握好。

雖然離開 Vue 了，不過我還是很喜歡 Vue 2，私心覺得它的寫法很優雅也容易上手，除了學習曲線平緩之外，官方甚至還結合 ESLint 推出 Style Guide，這個 Best Practice 可以讓新手有一個良好的開發準則參考。

因此 MapleStory Shop 這個 Side Project 我不會更新到 Vue 3 的 Composition API，而會繼續維護 Options API 的版本，至於之後其他的 Side Project 則是會使用 React 開發。

到目前為止，我也已經使用 React 實際開發專案半年了，算是逐漸熟悉這個生態圈了，同時也吸收了許多不同的知識，像是 Webpack、Formatter、Linter、CI/CD、Testing、Vulnerability 等等。近幾年前端領域愈來愈複雜了，期許自己之後慢慢朝向 DevOps for Frontend Developer 的方向邁進。

## 九、Windows -> Mac

年初購入了 MacBook M1 Pro 14″ (2021) 作為工作機！終於換掉了使用六年的 Windows 筆電，目前使用 Mac 也有一年了，我來說說幾個優點吧。

首先是我自己很在意的，那就是機身重量！這台 MacBook 比之前的 MSI 電競筆電輕太多了，電源轉接器也比較小巧。想到以前在學校扛那台微星龍走路到教室，還真的就像是扛一隻龍一樣，每天都在肌力訓練 💪

另外就是續航力不錯，可以持續 Coding 半天以上，不過我通常不會在外面 Coding，但是在 Demo 專案的時候不用擔心沒電，而且也不用一定要插著電才有高效能。另外，編譯效能的表現相當優秀，專案運行速度快了非常多。

總而言之，買來作為工作上的投資我覺得算是很有價值的。
