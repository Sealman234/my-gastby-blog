## 一、工作

今年離開樂屋，進入後續的 Funliday 與 Wistron 之後，算是確定從 Vue 轉換到 React 生態圈了。

我大概是從去年年底開始學習 React 的，由於只有下班與假日可以學習，所以花了將近五個月的時間，我才把 React 相關的技術都掌握起來。

話說回來，其實我很喜歡 Vue 2 的寫法，覺得很優雅也容易上手，所以 MapleStory Shop 那個專案不會更新到 Vue 3 Composition API 的寫法，而是會繼續維護下去。至於往後的 Side Project 則是轉而使用 React 開發哩。

目前已經使用 React 開發專案半年，算是逐漸熟悉這個生態圈了，同時也有吸收到 Webpack、Formatter、Linter、CI/CD、Testing、Vulnerability 等方面的知識。近幾年也有感受到前端領域愈來愈複雜，期許自己之後能慢慢朝向 DevOps For Frontend Developers 的方向邁進。
