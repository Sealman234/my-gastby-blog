## 一、工作

今年離開樂屋，後續進入 Funliday 與 Wistron 之後，算是確定從 Vue 轉換到 React 技術線了。

關於 React 我大約是從去年年底開始在 Udemy 上面看課程，由於只有下班後與假日有時間可以學習，又常常想休息耍廢打電動，所以拖拖拉拉地花了總共五個月的時間，才總算是把 React 生態圈的基本求職技能都掌握起來。

話說回來，其實我還是最喜歡 Vue 2 的寫法，覺得很優雅也容易上手，學習曲線平緩就算了，官方甚至結合 ESLint 推出 Style Guide，有了這個 Best Practice 的存在，就算是新手也能夠參考這個良好的開發準則！因此 MapleStory Shop 這個 Side Project 我應該不會更新到 Vue 3 的 Composition API，而是會繼續維護 Options API 的版本，至於之後其他的 Side Project 就使用 React 開發。

到目前為止已經使用 React 實際開發專案半年，算是逐漸熟悉這個生態圈了，同時也吸收了許多不同的知識，像是 Webpack、Formatter、Linter、CI/CD、Testing、Vulnerability 等等。近幾年前端領域愈來愈複雜，期許自己之後慢慢朝向 DevOps For Frontend Developer 的方向邁進。

## 九、Windows -> Mac

年初購入了 MacBook M1 Pro 14″ (2021) 作為工作機！終於換掉了使用六年的 Windows 筆電，目前使用 Mac 也有一年了，我來說說幾個優點吧。

首先是我自己很在意的，那就是機身重量！這台 MacBook 比之前的 MSI 電競筆電輕太多了，電源轉接器也比較小巧，想到以前在學校扛那台微星龍走路到教室，還真的就像是扛一隻龍一樣，每天都在肌力訓練 💪

另外就是續航力不錯，可以持續 Coding 半天以上，不過我通常不會在外面 Coding，但是在 Demo 專案的時候不用擔心沒電，而且也不用一定要插著電才有高效能，編譯效能的表現相當優秀，專案運行速度快了非常多。

總而言之，買來作為工作上的投資我覺得算是有價值的。
